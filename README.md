Quiz App

Link: https://efeg-simple-quiz-app.netlify.app/

[TR]

Bu proje, kullanıcıların sorulara cevap vererek bilgilerini test edebileceği basit bir quiz uygulamasıdır. Her soru için belirli bir süre tanınır ve kullanıcı doğru ya da yanlış cevap verdiğinde anında görsel geri bildirim alır. Tüm sorular tamamlandığında, toplam doğru sayısı ekranda gösterilir.

[EN]

This project is a simple quiz application where users can test their knowledge by answering questions. Each question has a countdown timer, and users receive instant visual feedback when they answer correctly or incorrectly. After completing all questions, the total number of correct answers is displayed on the screen.


Özellikler / Features

[TR]

- Her soru için geri sayım süresi (10 saniye)

- Doğru ve yanlış cevap geri bildirimi (renk ve ikonlar)

- Quiz sonunda tekrar oynama veya çıkma seçenekleri

- Toplam skor gösterimi

- Dinamik olarak soruların JavaScript üzerinden yüklenmesi

[EN]

- Countdown timer for each question (10 seconds)

- Real-time feedback with colors and icons

- Replay or quit options after the quiz

- Score summary at the end

- Dynamically generated questions from JavaScript

Kullanılan Teknolojiler / Built With

- HTML5

- CSS3 (Bootstrap 5 + Bootstrap Icons)

- JavaScript (Vanilla JS)
